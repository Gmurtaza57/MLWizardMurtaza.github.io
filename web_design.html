MY PORTFOLIO WEBSITE

📄 Structured Content: Divided the website into distinct sections, ensuring a logical content flow with clear demarcation of introduction, navigation, and main content.

🖱️ Integrated Navigation: Seamlessly embedded links, guiding users through the portfolio with ease and providing direct access to platforms like LinkedIn and GitHub.

🎨 Responsive Styling: Leveraged CSS3 to adaptively style elements, ensuring consistent visuals across varying screen sizes and devices.

✨ Visual Effects: Incorporated advanced CSS techniques to add transitions and visual effects, such as the parallax effect, enhancing the site's aesthetic appeal.

🔄 Dynamic Interaction: Used JavaScript and jQuery to introduce scrolling effects, dynamically revealing or hiding content based on user interactions and viewport positioning.

📏 Adaptive Features: Through JavaScript, implemented breakpoints to modify site behavior and display based on device screen sizes, guaranteeing an optimal user experience.

👤 About Me Section: Added a personal touch with a comprehensive 'About Me' section, offering visitors insights into personal experiences, aspirations, and professional journey.

HEROKU FACIAL EMOTION RECOGNITION APP

🖥️ Streamlined Backend: Utilized Flask as the web framework to route and serve the app. Integrated the main app functionality with `app.route('/')` to render the main HTML page.

📹 Real-time Emotion Detection: Incorporated OpenCV to capture video feed, detect faces, and predict emotions in real-time, offering a dynamic user experience.

🎨 Engaging UI: Created a user-friendly HTML interface that highlights the project's details, provides insights into detectable emotions, and integrates webcam functionality for users.

🔗 Seamless Interaction: Embedded direct links for users to view real-life emotion examples, aiding understanding and enhancing engagement.

📄 Deployment Readiness: Prepared the app for deployment with a Procfile and a comprehensive `requirements.txt`, ensuring all dependencies are clearly listed for smooth operation on Heroku.

🛠️ Robust Modelling: Integrated a pre-trained machine learning model to interpret facial expressions, ensuring accurate emotion predictions.

🌐 Webcam Integration: Employed JavaScript to allow users to activate their webcams directly from the browser, enhancing real-time emotion detection functionality.
